# Judul Project
Menganalisa faktor-faktor yang mempengaruhi nilai pasar (value) dari pemain sepakbola, seperti usia,performa, menit bermain, posisi, dan riwayat cedera.

## Repository Outline
1. P0M1_muhalifikri_dataset.csv - Dataset yang digunakan dalam analisa
2. P0M1_muhalifikri.ipynb - Notebook yang berisi pengolahan data dan hasil analisa 


## Problem Background

Sebuah tim sepakbola ingin mendatangkan pemain baru untuk season depan. Staf kepelatihan ingin agar pemain yang datang memiliki performa yang baik dan tidak mudah cedera. Namun, tim sepakbola ini belum mengetahui dulu faktor apa saja yang mempengaruhi harga/nilai pasar dari seorang pemain sepakbola. Sehingga tim manajemen pun bisa mengerti dan memahami pertimbangannya sampai dana pun lebih mudah untuk dikucurkan ke tim. 

## Project Output
Output dari project ini ialah suatu dashboard yang dapat digunakan untuk menganalisa nilai pasar(value) pemain sepak bola berdasarkan faktor yang mempengaruhinya.

link dashboard : https://public.tableau.com/app/profile/muhammad.ali.fikri5379/viz/milestone1p0w4-rev1/DashboardUtama?publish=yes

## Data
Sumber dataset : https://www.kaggle.com/datasets/khanghunhnguyntrng/football-players-transfer-fee-prediction-dataset

Identitas dataset : 10754 rows, 22 columns 

Content : berisikan data informasi para pemain sepak bola seperti performa (goals & assists), tinggi badan, riwayat cedera, posisi, jumlah kartu kuning, jumlah kartu merah, dsb.


## Stacks
Project ini menggunakan python dengan library pandas sebagai pengolah datanya, scipy sebagai pengolah data statistik, seaborn dan matplotlib sebagai visualisasi grafik.

dalam project ini juga menggunakan aplikasi Tableau untuk membuat Dashboard dari hasil pengolahan data.
